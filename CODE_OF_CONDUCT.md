# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The canonical text is maintained upstream at the link above; we do not fork it.

## Scope

The code of conduct applies in all project spaces: the GitHub repository (issues, pull requests, discussions, code review), mailing lists, chat channels, and any event where someone represents Sluice.

## Enforcement

Reports can be sent to `conduct@bino.bi`. Reports are handled confidentially by the maintainers listed in [GOVERNANCE.md](GOVERNANCE.md). We follow the Contributor Covenant's enforcement ladder (correction, warning, temporary ban, permanent ban).

If a report involves a maintainer, it is routed to the remaining maintainers; the subject of the report recuses themselves from any decision.
