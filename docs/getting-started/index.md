# Getting started

This track walks you from zero to a working local Sluice in roughly
thirty minutes.

1. [**Install**](install.md) — pick a binary, Docker image, or build from
   source.
2. [**Quickstart**](quickstart.md) — spin up the `hello-sluice` example
   (Docker Compose).
3. [**First query**](first-query.md) — run a masked, row-filtered query
   over `curl`, inspect the audit log.

If you prefer reading the concepts first, jump to
[Concepts → Policies](../concepts/policies.md) and come back here when
ready.
